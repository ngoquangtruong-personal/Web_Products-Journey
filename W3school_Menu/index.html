CSS Border Width


The border-width property specifies the width of the four borders.


The width can be set as a specific size (in px, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick:

Example


Demonstration of the different border widths:

            
p.one {
    border-style: solid;
    border-width: 5px;
}

p.two {
    border-style: solid;
    border-width: medium;
}

p.three {
    border-style: dotted;
    border-width: 2px;
}

p.four {
    border-style: dotted;
    border-width: thick;
  }
